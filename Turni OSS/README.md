# Hospital Shift Generator

An application that automatically generates work schedulers for Healthcare Support Workers (OOS)
based on configurable work cycles and scheduling rules
The project was developed to simulate the creation of hospital work schedules while respecting 
predefined constraints, demonstrating the use of object-oriented programming, modular design and algorithmic problem solving.

---

## Features
- Automatic generation of monthly work schedules
- Managment of multiple OSS employees
- Configurable work cycle
- Automatic assignment of rest days
- Rule-based scheduling logic
